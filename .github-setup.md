# GitHub Setup Instructions

Your Next.js surveillance web app is ready to push to GitHub!

## Quick Setup

### Step 1: Create GitHub Repository

Go to: **https://github.com/new**

Repository settings:
- **Name**: `camera-web`
- **Description**: "Browser-based surveillance camera with motion detection and recording"
- **Visibility**: Public (or Private if you prefer)
- **DO NOT** initialize with README, .gitignore, or license

Click "Create repository"

### Step 2: Push Your Code

Run these commands:

```bash
cd /Users/hadri/camera-web
git push -u origin main
```

That's it! Your code will be on GitHub.

### Step 3: Deploy to Vercel

1. Go to: **https://vercel.com/new**
2. Click "Import" next to your `camera-web` repository
3. Click "Deploy"
4. Wait ~1 minute
